# Poke-stimator PRD Doc
OVERVIEW:
A web app for Pokemon TCG collectors that quizzes their understanding of these products through a variety of estimation gamemodes. 

GOALS:
Allow users to demonstrate their knowledge through playing price guessing games, view prices from a volatile card market, and improve their understanding of a variety of cards across multiple eras, done in a manner that is easily accessible and fun.

USERS:
Pokemon TCG collectors, vendors, traders, investors, and hobbyists

FEATURES:
Card Browser by era and set
Upper-Lower Game
Trade Estimation Game
Cardle - Card Wordle

Tech Stack:
React, SupaBase, Python
- pulls from PokemonPriceTracker API
